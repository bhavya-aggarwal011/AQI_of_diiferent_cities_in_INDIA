# 📊 AQI Analysis of India

A comprehensive Power BI dashboard project that analyzes the Air Quality Index (AQI) across different cities in India. This project provides insights into pollution levels, city-wise air quality trends, pollutant concentrations, and overall environmental conditions using interactive visualizations.

---

## 📌 Project Overview

Air pollution has become one of the biggest environmental challenges in India. This dashboard helps analyze AQI data collected from multiple Indian cities to identify pollution patterns and support data-driven decision-making.

The project focuses on:

- City-wise AQI analysis
- Pollutant distribution
- AQI category comparison
- Time-based AQI trends
- Environmental insights through interactive dashboards
